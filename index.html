My Portfolio Website

Every web developer has to have his own portfolio website so I decided to create one for myself aswell.

This page started as a some kind of ,training' for me, but somewhere in the process I decided to make it my portfolio page.

I would divide this page into two parts. The first part, which consist of the header and the second part, which includes the rest of te body.

The difference between these two parts is the method used. The header is made with no external sources. Just pure html and css. The menu with help of JavaScript.

On the other hand I styled the main part of the page with Bootstrap 5 library.

I recently found out that the Bootstrap is a great tool, which makes the creating of the page much quicker and smoother, but I'm glad that I didn't use it before, because first of all it's important to learn the basics properly and make sure you are able to create it on your own.

Just for the record: I didn't use only Bootstrap 5. Everything is finished with custom css styling and some details with media queries.
